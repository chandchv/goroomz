# GoRoomz - Project Index

## 📋 Project Overview

**GoRoomz** is a comprehensive room booking platform that connects users with various types of accommodations across India. The platform features user authentication, room management, booking systems, and role-based access control for users, property owners, and administrators.

**Tech Stack:**
- **Frontend:** React 18 + Vite, Tailwind CSS, Radix UI, Framer Motion
- **Backend:** Node.js + Express.js
- **Database:** PostgreSQL with Sequelize ORM
- **Authentication:** JWT + Firebase (optional)
- **Deployment:** Supports production deployment with PM2
